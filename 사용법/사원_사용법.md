## 1. 회원가입

### 순서

1. 브라우저에서 시스템 주소(`http://localhost:8080` 또는 사내 URL)에 접속합니다.
2. 로그인 화면 하단의 **회원가입** 버튼을 클릭합니다.
3. **이메일 주소**를 입력합니다.
   - 반드시 회사 도메인 이메일이어야 합니다 (예: `홍길동@eactive.co.kr`).
   - 개인 이메일(gmail, naver 등)로는 가입할 수 없습니다.
4. **인증 코드 발송** 버튼을 클릭합니다. 입력한 이메일로 6자리 인증 코드가 전송됩니다.
   - 이메일이 오지 않으면 스팸함을 확인하거나, **재발송** 버튼을 클릭합니다.
5. 받은 인증 코드를 입력하고 **인증 확인**을 클릭합니다.
6. 나머지 정보를 입력합니다:
   - 이름 (실명)
   - 비밀번호 (8자 이상 권장)
   - 생년월일 (선택)
   - 전화번호 (선택)
7. **가입하기** 버튼을 클릭합니다.
8. 가입 완료 후 로그인 화면으로 이동합니다.

> **참고**: 이메일 인증 완료 즉시 계정과 개인 폴더가 자동으로 생성됩니다. 팀 배정이 필요한 경우 관리자에게 요청하세요.

## 2. 로그인 및 비밀번호 관리

### 로그인

1. 로그인 화면에서 **이메일**과 **비밀번호**를 입력합니다.
2. **로그인** 버튼을 클릭합니다.
3. 로그인 성공 시 대시보드로 이동합니다.

> 세션은 **30분 동안 활동이 없으면 자동 만료**됩니다. 만료 후에는 다시 로그인해야 합니다.

### 비밀번호 분실 시

1. 로그인 화면에서 **비밀번호를 잊으셨나요?** 링크를 클릭합니다.
2. 가입한 이메일 주소를 입력합니다.
3. 이메일로 전송된 재설정 코드를 입력합니다.
4. 새 비밀번호를 입력하고 변경합니다.

### 비밀번호 변경

1. 우측 상단 본인 이름 또는 **프로필** 메뉴로 이동합니다.
2. **비밀번호 변경** 섹션에서 현재 비밀번호와 새 비밀번호를 입력합니다.

## 3. 대시보드 둘러보기

로그인 후 가장 먼저 보이는 화면입니다.

| 영역 | 설명 |
|------|------|
| **내 폴더 바로가기** | 개인 문서 폴더로 이동합니다 |
| **최근 업로드** | 최근에 내가 올린 문서 목록 |
| **만료 예정 문서** | 30일 이내 만료되는 문서 알림 |
| **공용 폴더** | 전 직원이 공유하는 폴더 바로가기 |

## 4. 내 폴더 — 문서 관리

상단 내비게이션에서 **내 폴더**를 클릭합니다.

### 문서 목록 화면 구성

- **문서 카드**: 문서 제목, 종류, 검토 상태, 만료일이 표시됩니다.
- **검토 상태 배지**:

| 배지 | 의미 |
|------|------|
| 검토 대기 (노란색) | 관리자가 아직 검토하지 않은 상태 |
| 승인됨 (초록색) | 관리자가 승인한 문서 |
| 반려됨 (빨간색) | 관리자가 반려한 문서 (반려 사유 확인 필요) |

- **만료 배지**:

| 배지 | 의미 |
|------|------|
| 만료됨 (빨간색) | 만료일이 지난 문서 |
| 임박 (노란색) | 30일 이내 만료 예정 |

## 5. 문서 업로드

### 지원 파일 형식

| 형식 | 설명 |
|------|------|
| PDF | 미리보기 지원 |
| JPG, JPEG, PNG | 이미지로 미리보기 |
| DOCX | 미리보기 없음 (PDF 미리보기 파일을 별도 업로드 가능) |
| HWP, HWPX | 미리보기 없음 (PDF 미리보기 파일을 별도 업로드 가능) |
| PPT, PPTX | 미리보기 없음 |
| 기타 | 업로드 가능, 미리보기 없음 |

**최대 파일 크기**: 20MB

### 업로드 순서

1. **내 폴더** 화면에서 **새 문서 업로드** 버튼을 클릭합니다.
2. **문서 종류**를 선택합니다:
   - 이력서
   - 경력기술서
   - 졸업증명서
   - 정보처리기사 (자격증)
   - 건강보험료납부확인서
   - 증명사진
   - 기타
3. **문서 제목**을 입력합니다 (예: `2024년 이력서`).
4. **파일 선택** 버튼으로 파일을 선택합니다.
5. 필요 시 **만료일**을 설정합니다 (예: 자격증 유효기간 종료일).
6. **업로드** 버튼을 클릭합니다. 업로드 중에는 진행률 바가 표시되며, **업로드 취소** 버튼을 클릭하면 즉시 중단할 수 있습니다.

업로드 후 파일 크기에 따라 처리 방식이 다릅니다:

- **10MB 미만** — 자동 승인되어 즉시 사용 가능합니다.
- **10MB 이상** — **검토 대기** 상태가 되며, 관리자가 검토 후 승인 또는 반려합니다.

### 업로드 오류 안내

| 오류 메시지 | 원인 | 해결 방법 |
|-------------|------|-----------|
| 허용되지 않는 파일 형식 | 지원하지 않는 확장자 | 지원 형식 파일로 다시 업로드 |
| 파일 형식이 올바르지 않습니다 | 확장자를 변경한 파일 (예: .exe → .pdf 이름 변경) | 실제 해당 형식으로 저장된 파일 사용 |
| 동일한 문서가 이미 존재합니다 | 같은 폴더에 동일한 내용의 파일이 이미 있음 | 파일 내용이 다른지 확인 후 업로드 |
| 같은 종류와 제목의 문서가 이미 있습니다 | 폴더 내 동일 종류·제목 중복 | 제목을 다르게 입력 |

> **DOCX / HWP 미리보기**: 해당 형식은 브라우저에서 직접 미리볼 수 없습니다. 미리보기가 필요하다면 해당 문서를 PDF로 변환한 파일을 추가로 업로드하면 미리보기가 활성화됩니다.

## 6. 문서 상세 보기 및 미리보기

문서 목록에서 문서 제목 또는 **상세** 버튼을 클릭합니다.

### 상세 화면 구성

- **왼쪽 패널**: 문서 정보 (종류, 버전, 검토 상태, 파일명, 크기, 업로드일, 만료일)
- **오른쪽**: 문서 미리보기 (PDF는 뷰어로, 이미지는 직접 표시)

### 버전 관리

동일한 문서를 여러 번 업로드하면 버전이 자동으로 증가합니다.

- **v1**: 최초 업로드
- **v2**: 같은 문서 다시 업로드 시

버전 목록은 왼쪽 패널 하단에서 확인할 수 있습니다.

---

## 7. 문서 다운로드

문서 상세 화면 왼쪽 패널에서 **다운로드** 버튼을 클릭합니다.

- 승인된 문서뿐 아니라 검토 대기 중인 내 문서도 다운로드할 수 있습니다.
- 다운로드 시 원본 파일명으로 저장됩니다.

---

## 8. 만료일 설정

자격증이나 증명서처럼 유효기간이 있는 문서에 만료일을 설정할 수 있습니다.

1. 문서 상세 화면 왼쪽 **만료일 수정** 섹션에서 날짜를 선택합니다.
2. **저장** 버튼을 클릭합니다.
3. 날짜를 비우고 저장하면 만료일이 삭제됩니다.

만료일 30일 전부터 대시보드와 문서 목록에 경고 배지가 표시됩니다.

---

## 9. 문서 삭제

1. 문서 상세 화면 왼쪽 하단 **문서 삭제** 버튼을 클릭합니다.
2. 확인 팝업에서 **확인**을 클릭합니다.

> **주의**: 삭제 시 모든 버전과 파일이 **영구 삭제**됩니다. 복구가 불가능하므로 신중하게 진행하세요.

---

## 10. 공용 폴더 이용

전 직원이 공유하는 폴더입니다. 상단 내비게이션 **공용 폴더** 또는 대시보드 바로가기를 클릭합니다.

### 공용 폴더에서 할 수 있는 것

- 다른 직원이 올린 공용 문서 조회 및 다운로드
- 새 문서 업로드 (본인이 올린 문서에 한해 삭제 가능)

### 공용 폴더 문서 업로드

1. 공용 폴더 화면 하단 **새 문서 업로드** 버튼을 클릭합니다.
2. 문서 종류, 제목, 파일을 선택하고 업로드합니다.

> 공용 폴더에 올린 문서도 관리자의 검토 후 승인됩니다.

### 공유 폴더 이용 (권한 부여된 경우)

관리자가 특정 폴더 접근 권한을 부여한 경우, 상단 **공유 폴더** 메뉴에서 해당 폴더를 볼 수 있습니다.

## 12. 프로필 관리

상단 내비게이션에서 본인 이름 클릭 → **프로필** 또는 **내 정보** 메뉴로 이동합니다.

수정 가능한 항목:
- 전화번호
- 생년월일
- 비밀번호

> 이름, 이메일, 역할, 소속 팀, 직급은 관리자만 변경할 수 있습니다.

## 13. 자주 묻는 질문

**Q. 업로드한 문서가 '검토 대기' 상태입니다. 언제 승인되나요?**  
A. 파일 크기가 **10MB 이상**이면 관리자 검토가 필요합니다. 관리자가 검토 후 승인 또는 반려하며, 결과에 따라 이메일로 알림이 전송됩니다. 10MB 미만 파일은 업로드 즉시 자동 승인됩니다.

**Q. 문서가 반려되었어요. 어떻게 해야 하나요?**  
A. 문서 상세 화면에서 반려 사유를 확인한 후, 수정된 파일을 다시 업로드하세요. 새로운 버전으로 등록됩니다.

**Q. PDF가 아닌 파일을 올렸더니 미리보기가 없어요.**  
A. DOCX, HWP, HWPX, PPT 등은 브라우저 미리보기를 지원하지 않습니다. 해당 문서를 PDF로 변환한 후 '썸네일 재생성' 버튼을 누르거나, PDF 파일을 별도로 업로드하면 미리보기가 활성화됩니다.

**Q. 파일 크기 제한이 있나요?**  
A. 파일 하나당 최대 **20MB**입니다.

**Q. 로그인이 계속 풀립니다.**  
A. 세션은 **30분 비활동** 시 자동 만료됩니다. 장시간 작업 중이라면 주기적으로 페이지를 갱신하거나, 작업 후 다시 로그인하세요.

**Q. 회사 이메일이 아닌데 가입하고 싶어요.**  
A. 시스템은 회사 이메일 도메인(@eactive.co.kr)만 허용합니다. 다른 도메인이 필요하다면 관리자에게 문의하세요.
