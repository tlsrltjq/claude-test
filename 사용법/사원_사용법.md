# eActive Resource Hub 직원 사용 가이드

---

## 1. 회원가입

접속 주소의 로그인 화면 하단 **"회원가입"** 링크를 클릭합니다.

### 입력 항목

| 항목 | 필수 | 안내 |
|------|------|------|
| 이름 | 필수 | 실명 입력 (예: 홍길동) |
| 생년월일 | 필수 | `2001.09.04` 형식으로 입력 |
| 연락처 | 필수 | `010-0000-0000` 형식 |
| 회사 이메일 | 필수 | `@eactive.co.kr` 앞부분만 입력 (예: `hong`) |
| 아이디 | 필수 | 로그인에 사용할 영문/숫자 ID |
| 팀 | 선택 | 소속 팀을 목록에서 선택 |
| 직급 | 필수 | 목록에서 본인 직급 선택 |
| 비밀번호 | 필수 | 영문·숫자·특수문자 포함 8자 이상 |
| 비밀번호 확인 | 필수 | 위와 동일하게 재입력 |

### 가입 후 절차

1. **이메일 인증** — 입력한 회사 이메일로 인증 메일이 발송됩니다. 메일 내 링크를 클릭해 인증을 완료합니다.
2. **관리자 승인 대기** — 인증 완료 후 관리자가 계정을 활성화해야 로그인이 가능합니다. 승인 전까지는 로그인 시 "승인 대기" 상태로 표시됩니다.
3. **승인 완료** — 관리자 승인 후 정상 로그인이 가능합니다. 승인 관련 문의는 `shingi@eactive.co.kr`로 연락하세요.

---

## 2. 로그인

- **로그인 방법**: 아이디(또는 이메일) + 비밀번호 입력
- **비밀번호를 잊었을 때**: 로그인 화면의 **"비밀번호를 잊으셨나요?"** 링크 → 회사 이메일 입력 → 메일로 인증 코드 수신 → 새 비밀번호 설정

---

## 3. 대시보드

로그인 직후 이동하는 메인 화면입니다.

- **왼쪽 프로필 카드**: 이름, 이메일, 권한, 팀, 직급, 생년월일, 연락처, 개발자 등급이 표시됩니다.
- **오른쪽 메뉴 그리드**: 자주 쓰는 기능으로 바로 이동할 수 있습니다.

---

## 4. 내 폴더

대시보드 → **내 폴더** 클릭.

업로드한 문서가 카드 형태로 표시됩니다.

### 화면 구성

- **양식 이력서 배너** (등록된 경우): 관리자가 발급한 양식 이력서가 있으면 상단에 표시됩니다. **다운로드** 버튼으로 즉시 받을 수 있습니다.
- **문서 카드**: 문서 종류 배지, 검토 상태(검토 대기 / 승인됨 / 반려됨), 만료 상태(만료됨 / 만료임박), 파일 썸네일(PDF·이미지), 버전 번호, 업로드 날짜가 표시됩니다.
- **카드 하단 버튼**:
  - 다운로드 아이콘: 현재 버전 파일을 즉시 다운로드
  - `⋯` 아이콘: 문서 상세 페이지로 이동

### 필터 / 검색 / 정렬

- **종류 필터**: 드롭다운으로 문서 종류를 선택해 해당 종류만 표시
- **검색**: 제목 또는 파일명으로 실시간 검색
- **정렬**: 최근 업로드 / 종류 / 제목 순으로 정렬

---

## 5. 문서 업로드

내 폴더 상단 **"문서 업로드"** 버튼 클릭.

### 입력 절차

**① 문서 종류 선택** (필수)
드롭다운에서 해당하는 종류를 선택합니다. 선택하면 제목 입력칸의 예시 문구가 자동으로 바뀝니다.

**② 문서 제목 입력** (필수)
예시 문구를 참고해 알아보기 쉬운 제목을 입력합니다.

**③ 추가 정보 입력** (졸업증명서·정보처리기사만 해당)
- **졸업증명서**: 학위 종류(고졸/전문학사/학사/석사/박사)와 졸업일을 추가로 입력합니다.
- **정보처리기사**: 자격증 취득일을 추가로 입력합니다.

**④ 파일 선택**
- 영역을 클릭하거나 파일을 끌어다 놓으면 첨부됩니다.
- **지원 형식**: PDF, JPG, JPEG, PNG, DOCX, HWP, HWPX, PPT, PPTX
- **최대 크기**: 20MB (10MB 초과 시 관리자 승인 후 공개)

**⑤ 만료일 입력** (선택)
건강보험 관련 서류처럼 유효기간이 있는 문서는 만료일을 입력하면 기간 경과 시 자동으로 "만료됨" 배지가 표시됩니다.

**⑥ 업로드 버튼 클릭**
진행률 바가 표시되며 완료 후 자동으로 내 폴더로 이동합니다. 업로드 중에는 **"업로드 취소"** 버튼으로 중단할 수 있습니다.

### 업로드해야 할 자료 목록

아래 서류를 순서대로 업로드해두면 인력 표에 자동으로 반영됩니다.

| 문서 종류 | 설명 | 비고 |
|-----------|------|------|
| **이력서** | 경력 기술이 포함된 이력서 | 양식 이력서는 관리자에게 문의 |
| **SW기술자 경력증명서** | 한국소프트웨어산업협회 발급 경력증명서 | |
| **졸업증명서** | 최종 학력 졸업증명서 | 학위 종류·졸업일 입력 필요 |
| **정보처리기사** | 자격증 사본 | 취득일 입력 필요 |
| **국민연금가입증명서** | 국민건강보험공단 또는 국민연금공단 발급 | |
| **건강보험가입증명서** | 국민건강보험공단 발급 | |
| **건강보험자격득실확인서** | 국민건강보험공단 발급 | |
| **증명사진** | JPG/PNG 증명사진 파일 | |
| **서명** | 본인 서명 이미지 | JPG/PNG |
| **기타** | 위 항목에 해당하지 않는 서류 | |

> **새 버전 업로드**: 이미 업로드된 문서를 다시 업로드하면 새 버전으로 저장됩니다. 이전 버전은 상세 페이지에서 확인할 수 있습니다.

## 6. 문서 상세 페이지

내 폴더에서 카드의 **`⋯`** 버튼 클릭 시 이동.

### 왼쪽 패널

- **문서 정보**: 종류, 현재 버전 번호, 검토 상태, 반려 사유(반려된 경우), 파일명, 크기, 업로드 날짜, 만료일
- **검토 상태 의미**:
  - `검토 대기`: 관리자가 아직 확인하지 않은 상태
  - `승인됨`: 관리자가 정상 서류로 확인한 상태
  - `반려됨`: 서류에 문제가 있어 재업로드가 필요한 상태. 반려 사유를 확인 후 수정하여 다시 업로드하세요.
- **다운로드** 버튼: 현재 버전 파일을 다운로드
- **만료일 수정**: 날짜를 변경하고 **저장** 버튼 클릭. 비워두면 만료일 없음으로 처리됩니다.
- **버전 목록**: 해당 문서의 전체 업로드 이력이 표시됩니다.
- **문서 삭제**: 모든 버전과 파일이 영구 삭제됩니다. 삭제 전 확인창이 표시됩니다.

### 오른쪽 패널

- **미리보기**: PDF·이미지 파일은 바로 미리볼 수 있습니다. DOCX·HWP·HWPX는 미리보기가 지원되지 않으며, 다운로드 후 확인하세요.

---

## 7. 문서 검색

대시보드 → **문서 검색** 클릭.

본인이 접근 권한을 가진 문서를 키워드로 검색할 수 있습니다.

---

## 8. 공용 폴더 / 공유 폴더

대시보드의 **공유** 섹션에서 이동합니다.

- **공용 폴더**: 전 직원이 열람 가능한 공통 자료가 보관되어 있습니다.
- **공유 폴더**: 특정 팀 또는 그룹이 공유하는 폴더입니다. 관리자가 접근 권한을 부여한 폴더만 표시됩니다.

---

## 9. 다운로드 이력

대시보드 → **다운로드 이력** 클릭.

본인이 다운로드한 파일의 이력이 날짜·IP 주소와 함께 표시됩니다.

---

## 10. 설정

대시보드 → **설정** 클릭. 탭 3개로 구성됩니다.

### 계정 정보 탭
현재 이름, 이메일, 아이디, 권한, 상태, 팀, 직급, 연락처, 생년월일을 확인할 수 있습니다.

### 개인정보 수정 탭
- **수정 가능**: 연락처, 생년월일
- **수정 불가** (관리자에게 문의): 이름, 이메일, 팀, 직급

### 비밀번호 변경 탭
현재 비밀번호 입력 → 새 비밀번호 입력 → 새 비밀번호 확인 → **변경** 버튼 클릭.
