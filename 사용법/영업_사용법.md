## 1. 영업 역할 개요

영업(SALES) 역할은 사내 인력 데이터를 영업 업무에 활용하기 위해 부여됩니다.

**사원과의 차이점:**

| 기능 | 사원 | 영업 |
|------|------|------|
| 본인 폴더 관리 | O | O |
| 전 직원 프로필 조회 | X | O |
| 전 직원 문서 열람 | X | O (승인된 문서만) |
| 전 직원 문서 다운로드 | X | O |
| 경력 계산기 | X | O |
| 프로필 엑셀 내보내기 | X | O |
| 문서 검토·승인 | X | X (관리자만) |
| 직원 계정 관리 | X | X (관리자만) |

---

## 2. 로그인 및 대시보드

로그인 방법은 사원과 동일합니다. 로그인 후 대시보드에 **영업 메뉴**가 추가로 표시됩니다.

대시보드 상단 내비게이션:
- **대시보드**: 메인 화면
- **내 폴더**: 본인 문서 관리
- **인력 현황**: 전 직원 목록 및 프로필
- **경력 계산기**: 경력 기간 및 단가 계산
- **공용 폴더**: 전 직원 공용 문서

---

## 3. 인력 현황 조회

상단 내비게이션 **인력 현황** (또는 `/sales/employees`) 메뉴를 클릭합니다.

### 직원 목록 화면

전 직원의 기본 정보가 카드 또는 테이블 형태로 표시됩니다.

| 표시 항목 | 설명 |
|-----------|------|
| 이름 | 직원 실명 |
| 직급 | 대표 / 전무 / 상무 / 이사 / 부장 / 차장 / 과장 / 대리 / 사원 |
| 소속 팀 | 배정된 팀명 |
| 역할 | 사원 / 영업 / 관리자 |

### 검색 및 필터

- **이름 검색**: 이름 일부를 입력하면 실시간 필터링됩니다.
- **팀 필터**: 드롭다운으로 특정 팀만 조회합니다.
- **직급 필터**: 특정 직급만 조회합니다.
- **정렬**: 이름 순, 팀 순, 직급 순으로 정렬 가능합니다.

### 엑셀 내보내기

목록 화면 우측 상단 **엑셀 내보내기** 버튼을 클릭하면 현재 필터 조건에 맞는 직원 목록이 `.xlsx` 파일로 다운로드됩니다.

포함 항목: 이름, 직급, 팀, 역할, 입사일, 경력 정보 등

---

## 4. 직원 프로필 상세 보기

직원 목록에서 이름을 클릭하면 해당 직원의 프로필 상세 화면으로 이동합니다.

### 프로필 상세 화면 구성

| 섹션 | 내용 |
|------|------|
| 기본 정보 | 이름, 직급, 팀, 역할, 전화번호 |
| 경력 정보 | 입사일, 총 경력 기간, 이전 경력 |
| 학력 | 졸업증명서 기반 최종 학력 |
| 자격증 | 등록된 자격증 정보 |
| 문서 목록 | 승인된 문서 목록 (이력서, 경력기술서 등) |

---

## 5. 직원 문서 열람 및 다운로드

직원 프로필 상세 화면 하단 **문서 탭** 또는 `/sales/employees/{직원ID}/documents`에서 확인합니다.

### 열람 가능한 문서

**승인(APPROVED)** 상태인 문서만 열람할 수 있습니다. 검토 대기 중이거나 반려된 문서는 표시되지 않습니다.

### 문서 미리보기

문서 카드의 **미리보기** 버튼 또는 썸네일 이미지를 클릭하면 새 탭에서 미리보기가 열립니다.

- PDF: 브라우저 내장 PDF 뷰어로 표시
- 이미지(JPG, PNG): 이미지로 표시
- DOCX, HWP 등: 미리보기 없음 (다운로드만 가능)

### 문서 다운로드

**다운로드** 버튼을 클릭하면 원본 파일이 저장됩니다. 모든 다운로드 내역은 감사 로그에 기록됩니다.

---

## 6. 프로필 엑셀 내보내기

전 직원의 프로필 데이터를 엑셀 파일로 내보냅니다.

1. 상단 내비게이션 **인력 현황** → 우측 상단 **엑셀 내보내기** 버튼 클릭
2. 또는 직접 `/sales/profiles/export` 접근

엑셀에는 다음 항목이 포함됩니다:
- 이름, 직급, 소속 팀
- 입사일, 총 경력 기간
- 최종 학력, 학위 취득일
- 자격증명, 취득일
- 전화번호

> 미등록 항목은 빈 칸으로 내보내집니다.

## 7. 경력 계산기

상단 내비게이션 **경력 계산기** (또는 `/sales/career-calculator`) 메뉴를 클릭합니다.

### 기능 개요

특정 직원의 경력 기간을 계산하고 단가를 산출하는 도구입니다.

### 사용 방법

#### 자동 입력 (추천)

1. **직원 선택** 드롭다운에서 이름을 고릅니다.
2. **자동 입력** 버튼을 클릭합니다.
3. 해당 직원의 등록된 정보(최종 학력, 자격증 등)가 자동으로 채워집니다.
   - 여러 학위가 있는 경우 **가장 높은 학위**가 자동 선택됩니다.

#### 수동 입력

각 항목을 직접 입력합니다:

| 항목 | 설명 |
|------|------|
| 입사일 | 해당 직원의 입사일 |
| 계산 기준일 | 경력을 계산할 기준 날짜 (기본값: 오늘) |
| 최종 학위 | 전문학사 / 학사 / 석사 / 박사 |
| 졸업일 | 최종 학위 취득일 |
| 자격증명 | 보유 자격증 (예: 정보처리기사) |
| 자격증 취득일 | 해당 자격증 취득일 |
| 이전 경력 기간 | 현 회사 입사 전 타 직장 경력 (월 단위) |

#### 계산 결과

**계산하기** 버튼을 클릭하면:
- 총 경력 연수/월수
- 경력 기반 단가 (회사 단가표 적용)
- 학위 보너스, 자격증 보너스 항목별 내역

#### 결과 저장

계산 결과 화면에서 **저장** 버튼을 클릭하면 해당 직원의 프로필에 경력 정보가 저장됩니다.

## 9. 자주 묻는 질문

**Q. 특정 직원의 문서가 보이지 않아요.**  
A. 해당 문서가 아직 **검토 대기** 중이거나 **반려** 상태인 경우 영업 계정에서는 보이지 않습니다. 관리자에게 문서 승인 요청을 하거나, 직원에게 재업로드를 요청하세요.

**Q. 엑셀 내보내기 시 일부 직원 정보가 비어 있어요.**  
A. 해당 직원이 졸업증명서나 자격증 등을 아직 업로드하지 않았거나, 업로드한 문서가 승인되지 않은 경우입니다. 직원에게 문서 등록을 요청하세요.

**Q. 경력 계산기에서 자동 입력이 안 돼요.**  
A. 해당 직원이 졸업증명서 또는 자격증을 업로드하지 않았거나, 문서가 승인되지 않은 경우 자동 입력이 동작하지 않습니다. 수동으로 입력하거나 관리자에게 문의하세요.

**Q. 문서를 다운로드할 수 없어요.**  
A. 문서 상태가 **승인됨**이어야 다운로드할 수 있습니다. 관리자에게 문서 승인을 요청하세요.

**Q. 경력 계산 결과가 예상과 달라요.**  
A. 입사일, 이전 경력, 졸업일이 정확히 입력되었는지 확인하세요. 계산 기준일도 오늘 날짜로 설정되어 있는지 확인합니다.
