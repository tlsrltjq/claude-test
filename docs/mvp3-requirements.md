# MVP3 — 요구사항 (사용자 원본 정리본)

> 사용자가 보낸 후속 작업 목록을 화면별로 분류한 것. 의역은 최소화하고 의도가 모호한 부분은 `❓`로 표시.

## /login

- **ID 저장 체크박스 추가**
  - 체크 시 이메일 입력칸에 로그인 정보 저장 (다음 방문 시 이메일 미리 채움)
  - 체크 해제 시 비밀번호만 입력해서 들어갈 수 있게 함
  - ❓ "비밀번호만 치면 들어갈 수 있게" — 이건 자동 로그인이 아니라 이메일이 미리 들어가 있으니 비밀번호 한 줄만 치면 된다는 뜻으로 해석. (Remember-me 비밀번호 저장은 보안상 금지)
- **비밀번호 찾기 기능 추가**
  - 가입된 이메일 입력 → 인증번호 → 확인 후 랜덤 문자열 이메일로 발송

## /signup

- 입력 폼 순서: **이름 → 생년월일 → 연락처 → 회사 이메일 → 팀 선택 → 직급 선택 → 비밀번호**
- 이메일 잘못 입력 시 비밀번호 초기화하지 않기 (현 상태 유지 — 검증 항목)
- 직급 선택에 **상무** 옵션 추가
- 생년월일 8자로 입력 (예: `20010904`)

## /signup/resend (이메일 인증 재발송 화면)

- 이메일 인증 시간 5분으로 변경, 유효시간 5분 설정
- 5분 타이머 추가

## /dashboard

- 내 정보에 추가 필드: 개발자 등급, 생년월일, 본인 팀 등 간단한 추가 정보 삽입

## /myfolder

- 문서 상세에 **태그 기능 삭제** (이제 태그 사용 안 함)
- 문서 업로드:
  - 문서 종류: **재직증명서 삭제, 증명사진 추가, 자격증 → 정보처리기사로 이름 변경**
  - 파일 업로드 허용 형식에 **ppt, pptx 추가**
- **내문서 삭제 기능 추가**

## /search

- 들어가면 아무것도 안 나옴 → **들어갔을 때 본인 권한의 모든 문서가 바로 표시되게**
- 표시 범위: 내 폴더 + 공용 폴더 + 권한 받은 사용자 폴더 모든 파일
- 태그 기능 삭제 (모든 화면에서)
- 그 다음에 **검색 기능을 활용한 필터** 추가

## /shared/folders

- **전 사원 공용 폴더 추가**
  - 용도: 필요한 문서 업로드 가이드, 발급 가이드, 영업 신규 프로젝트 진입용 이력서 양식 업데이트 등
  - 모두 업로드 가능, 다운로드 가능
  - 삭제는 **관리자**와 **본인이 올린 문서**만 가능

## /sales/members

- 전사 인력 조회에서 **정렬 기능** 추가
- 기본 = 직급별 정렬
- 팀 컬럼 클릭 → 팀별 정렬
- 역할 컬럼 클릭 → 역할별 정렬

## /sales/profiles

- 개발자 등급별 인원 체크 표 작게 하나 만들기 (요약 위젯)
- 필요없는 컬럼 컨펌 받아 삭제 + 추가할 것들 다시 정리
- **사용자 정의 컬럼 옵션** — 사용했던 컬럼 조합을 저장해서 재사용
- 경력 3가지 표시 방식
  - `n년 n월 n일`
  - `n개월`
  - `n일`
- **체크 표시** 만들기 → 체크된 인원만 엑셀로 내보내기 (mvp2 08-excel-export 흡수)

## /sales/career-calculator

- 검색 기능 점검 (현재 동작 X)
- 오른쪽에 검색 버튼 추가 + Enter 키로도 검색 가능
- ❓ "선택한 사람 경력 불러오기"는 **나중에** 할 기능 (이번 라운드 제외)

## 관리자 대시보드

- 관리자용 직원 목록에도 **검색 기능** 추가 (이름, 직급, 권한, 팀)
- 직급 한글로 작성
- **개인폴더 체크 컬럼 삭제** (이제 모두가 개인폴더를 가짐)
- 직원 상세 페이지에서 **계정 활성/비활성 버튼** 추가 (비활성 시 모든 권한 박탈, 로그인도 안 됨)
- **권한 네이밍 변경** (관리자 / 영업 / 사원)
