# eActive Resource Hub 영업 사용 가이드

> 직원 공통 기능(로그인, 내 폴더, 문서 업로드, 설정 등)은 **직원 사용 가이드** 참조.
> 이 문서는 영업(SALES) 권한에서 추가로 사용할 수 있는 기능만 설명합니다.

---

## 1. 전사 인력 조회

대시보드 → **전사 인력 조회** 클릭 (`/sales/members`).

회사 전체 직원 목록이 표시됩니다. **영업본부·경영본부 포함 전 팀**이 나타납니다.

### 검색 / 필터

- **이름 또는 이메일** 키워드 입력
- **팀** 드롭다운으로 특정 팀만 표시
- **검색** 버튼 클릭, **초기화** 버튼으로 전체 목록 복원

### 정렬

테이블 헤더(이름 / 팀 / 직급 / 역할)를 클릭하면 오름차순·내림차순으로 정렬됩니다. 현재 정렬 방향은 ↑↓ 화살표로 표시됩니다.

### 문서 보기

각 직원 행 오른쪽 **문서** 버튼을 클릭하면 해당 직원이 업로드한 문서 목록을 조회하고 다운로드할 수 있습니다.

---

## 2. 인력표

대시보드 → **인력 표** 클릭 (`/sales/profiles`).

프로젝트에 투입 가능한 팀(관리자가 설정한 팀)의 직원 정보와 보유 문서 현황을 한눈에 볼 수 있는 표입니다.

### 등급별 현황 요약

상단에 특급 / 고급 / 중급 / 초급별 인원 수가 카드로 표시됩니다.

### 검색 / 필터

| 항목 | 설명 |
|------|------|
| 이름·이메일 | 키워드 검색 |
| 팀 | 복수 선택 가능 (드롭다운 체크박스) |
| 직급 | 단일 선택 |
| 개발자 등급 | 복수 선택 가능 (특급/고급/중급/초급) |
| 경력 표시 | 년월 / 개월 / 일 중 선택 (변경 즉시 적용) |

**검색** 버튼으로 적용, **초기화** 버튼으로 전체 조건 해제.

### 컬럼 설정

툴바의 **컬럼 설정** 버튼 클릭 시 패널이 열립니다. 체크박스로 표에 표시할 컬럼을 선택합니다. 선택 가능한 컬럼:

팀 / 직급 / 나이 / 생년월일 / 연락처 / 이메일 / 개발자 등급 / 경력 / 이력서 / SW기술자 경력증명서 / 정보처리기사 / 졸업증명서 / 국민연금가입증명서 / 건강보험가입증명서 / 건강보험자격득실확인서 / 기타자료

### 엑셀 다운로드

- **전체 엑셀**: 현재 필터·컬럼 설정 기준으로 전체 행을 `.xlsx`로 다운로드
- **선택 엑셀**: 표에서 체크박스로 행을 선택한 후 클릭. 선택 인원 수가 버튼에 표시됩니다.

### ZIP 묶음 다운로드

표에서 체크박스로 직원을 선택한 후 **ZIP 묶음** 버튼 클릭. 선택한 직원들의 문서 파일을 ZIP 압축 파일 하나로 다운로드합니다.

### 프리셋

자주 쓰는 필터·컬럼 조합을 저장해두고 빠르게 불러올 수 있습니다.

- **저장**: 툴바 우측 **저장** 버튼 클릭 → 프리셋 이름 입력 → 저장. 현재 컬럼 설정, 정렬, 경력 표시 형식이 함께 저장됩니다.
- **불러오기**: **프리셋 선택** 드롭다운에서 저장된 프리셋을 선택하면 즉시 적용됩니다.

### 정렬

표 헤더(이름 / 팀 / 직급 / 개발자 등급 / 경력)를 클릭해 정렬합니다. 다시 클릭하면 역순으로 바뀝니다.

### 문서 조회

이름 링크를 클릭하면 해당 직원의 문서 목록 페이지로 이동해 개별 파일을 다운로드할 수 있습니다.

## 3. 경력 계산기 / 기술등급 산출

대시보드 → **경력 계산기** 클릭 (`/sales/career-calculator`).

학위·자격증 기준으로 경력 기간을 입력하면 SW 기술자 등급(특급/고급/중급/초급)을 자동으로 산출합니다.

### 사용 순서

**① 직원 선택** (선택 사항)

왼쪽 직원 목록에서 이름을 클릭해 대상 직원을 선택합니다. 이름이나 팀으로 검색할 수 있습니다.

직원을 선택하면 해당 직원이 업로드한 졸업증명서·자격증 정보가 아래 기본 정보 항목에 **자동으로 입력**됩니다.

**② 기본 정보 입력**

| 항목 | 설명 |
|------|------|
| 학위 | 고졸 / 전문학사 / 학사 / 석사 / 박사 중 선택 |
| 졸업일자 | 학력 기반 경력 기산점 |
| 정보처리기사 | 체크박스로 보유 여부 선택, 체크 시 취득일자 입력 |

> **등급 기준 참고표** 버튼을 클릭하면 학위별·자격증별 등급 기준(필요 경력 연수)을 확인할 수 있습니다.

**③ 경력 기간 입력**

| 컬럼 | 설명 |
|------|------|
| 소속회사 | 경력 회사명 (선택 입력) |
| 시작일자 | 해당 회사 입사일 |
| 종료일자 | 퇴사일 또는 현재 재직 중인 경우 오늘 날짜 |

- **기간 추가** 버튼으로 행을 추가합니다.
- 날짜를 입력하면 **일수**와 **중복 여부**가 즉시 계산됩니다.
- 기간이 겹치는 행은 노란색으로 강조되며 "중복"으로 표시됩니다.
- **중복 기간 자동 제거** 체크박스를 활성화하면 서버에서 중복 구간을 자동으로 제거해 계산합니다.
- 행 오른쪽 **−** 버튼으로 해당 행을 삭제합니다.

**④ 등급 산출**

**등급 산출** 버튼 클릭. 오른쪽 패널에 결과가 표시됩니다.

### 결과 패널 구성

| 항목 | 설명 |
|------|------|
| 최종 제안등급 | 학위 기준과 자격증 기준 중 높은 등급을 최종 등급으로 제시 |
| 기준별 경력 비교 | 학위 기준 / 자격증 기준 각각의 인정일수, 경력기간, 환산경력, 적용등급 |
| 경력 기간 상세 | 입력한 각 구간의 총일수, 학력인정일수, 자격증인정일수, 중복 여부 |

### 프로필 저장

결과 패널의 **프로필 저장** 섹션에서 직원을 선택한 상태라면 개발자 등급을 확인하거나 수정한 뒤 **프로필에 저장** 버튼을 클릭합니다. 저장된 등급은 인력표와 대시보드의 개발자 등급 항목에 반영됩니다.

> 직원을 선택하지 않은 상태에서는 저장 버튼이 활성화되지 않습니다.
