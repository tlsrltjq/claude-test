파일 GC (Garbage Collection)

소프트 삭제된 문서의 물리 파일을 영구 제거합니다. 매일 새벽 2시 자동 실행됩니다.

GC 정책
보존 기간
자동 실행 매일 새벽 02:00
대상 조건 status = DELETED
deleted_at < now() − 보존기간
files_purged_at IS NULL
삭제 대상 각 버전의 원본 파일 + 미리보기 파일
보존 기간 변경 환경변수 RESOURCEHUB_GC_RETENTION_DAYS
수동 실행
주의: 수동 실행 시 삭제된 파일은 복구할 수 없습니다. 보존 기간(일)이 지난 문서의 물리 파일만 삭제됩니다.

GC를 지금 즉시 실행합니다. 조건을 충족하는 문서가 없으면 아무 작업도 수행하지 않습니다.

  • 개별 파일 삭제 실패는 경고 로그만 남기고 계속 진행됩니다.
  • 처리 완료된 문서는 files_purged_at이 기록됩니다.
